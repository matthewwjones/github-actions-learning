# github-actions-learning

A simple repo to support GH Actions learning exercises.

## Clone

```bash
$ git clone git@github.com:matthewwjones/github-actions-learning.git
```

## Build

```bash
$ cd github-actions-learning
$ ./gradlew build 
```

## Run

```bash
$ ./gradlew bootRun 
```

Then point your browser at:

* http://localhost:8080
